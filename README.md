
## **Herencia Java Ejercicio** ##

Peque√±o ejercicio de herencia en Java propuesto para entrevista.
**Descarga**:
```
git clone https://github.com/lorenmanu/herenciaJava.git
cd herenciaJava
```
**Compilacion**:
```
clear && rm -r src/*/*.class
javac src/Figura/*.java src/Main/*.java
```
**Ejecucion**:
```
cd src
java Main.Main
```
